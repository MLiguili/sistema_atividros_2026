# Sistema Atividros

ERP para empresas de molduras, vidros e quadros.

## Stack
- Backend: FastAPI
- Frontend: React + Vite
- Banco: PostgreSQL (Supabase)
- ORM: SQLAlchemy
- Migrações: Alembic
- Autenticação: JWT

## Módulos
- Clientes
- Pedidos
- Estoque
- Tabela de preços
- Produção
- Financeiro (planejado)
