# Regras de Negócio

## Pedido
Status:
Orçamento → Produção → Finalizado → Entregue.

Cada pedido possui:
- Cliente
- Itens
- Materiais
- Valor
- Datas

## Estoque
A produção deve baixar automaticamente:
- Moldura
- Vidro
- Fundo
- Paspatour
- Insumos

## Objetivo
Todo cálculo deve ser reproduzível e auditável.
